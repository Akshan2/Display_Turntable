# Display Turntable

Sup folks!

Required components will be needed to recreate this project :

- 1 * Microcontroller (ESP 32, ESP 8826, Arduino Uno/Nano/ Mega, etc)
- 1 * NEMA 17 Stepper Motor
- 1 * L298N 12V 2A Motor Driver
- 1 *  AC to DC 12V @ 2A Power Supply
- 1 * 10k ohm Potentiometer
- Few Jumper Wires
